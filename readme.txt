== Description ==

Spotlighter is a cool tool that will help you to divert the readers attention in various ways. With Spotlighter you can convert the content to Modal-esque Boxes & Highlighted Text.
 
Creating the spotlights is pretty simple, and requires no prior knowledge in any coding language. Its done using simple [spotlight] shortcode.

Spotlighter creates wrappers around its elements, so it does not affect the original content, which maintains whatever structure or design it had (apart from using children selectors on CSS3 or JS).

Spotlighter a powerful tool to emphasize content in a dynamic way. 

**On the current version, you can show :**
 
* Simple Highlight / Spotlight.
* Completely control the design of the Spotlight using either shortcode attributes or plain css.
* Multi-Highlight / Spotlight using a single Switch.
* Spotlight External Elements.
* Drop shadow Support.

**Coming Soon :**

* Presets.
* Animated Effects.

You can find detailed information with examples in [Spotlighter](http://dev.nimrodtsabari.net/wp/spotlighter-a-wordpress-plugin/ "Spotlighter's Page").

== Installation ==

1. Upload the plugin zip content to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Thats it!
